## Descripción

Este repositorio contiene una colección de **laboratorios prácticos de ciberseguridad** enfocados en seguridad ofensiva, seguridad inalámbrica y ataques basados en hardware.

El propósito de estos laboratorios es practicar y documentar **técnicas de seguridad utilizadas en entornos reales dentro de un ambiente controlado**. Cada laboratorio demuestra cómo funcionan ciertos ataques y cómo pueden detectarse o mitigarse.

Estos proyectos forman parte de mi **proceso de aprendizaje en ciberseguridad** y están diseñados para documentar experiencia práctica con herramientas de seguridad, conceptos de redes y metodologías de hacking ético.

---

## ⚠️ Aviso Ético

Todos los experimentos documentados en este repositorio fueron realizados **en entornos de laboratorio controlados**, utilizando sistemas y redes creados específicamente para pruebas.

Las técnicas demostradas aquí son comúnmente utilizadas por **profesionales de ciberseguridad y testers de penetración** para evaluar la seguridad de sistemas y redes.

Estos métodos deben utilizarse **únicamente en sistemas o redes de tu propiedad o en aquellos donde tengas autorización explícita para realizar pruebas de seguridad**.

El acceso no autorizado a sistemas informáticos o redes inalámbricas puede ser ilegal y podría resultar en **consecuencias legales graves**.

Este repositorio tiene fines **estrictamente educativos y de investigación ética en ciberseguridad**.
