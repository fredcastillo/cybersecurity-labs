## Overview

This repository contains a collection of **hands-on cybersecurity laboratories** focused on offensive security, wireless security, and hardware-based attacks.

The purpose of these labs is to practice and document **real-world security techniques in a controlled environment**. Each lab demonstrates how specific attacks work and how they can be detected or mitigated.

These projects are part of my **cybersecurity learning journey** and are intended to document practical experience with security tools, networking concepts, and ethical hacking methodologies.

---

## ⚠️ Ethical Notice

All experiments documented in this repository were conducted **in controlled lab environments** using systems and networks specifically created for testing purposes.

The techniques demonstrated here are commonly used by **security professionals and penetration testers** to evaluate the strength of systems and network defenses.

These methods must **only be used on systems or networks that you own or have explicit permission to test**.

Unauthorized access to computer systems or wireless networks without permission may be illegal and could result in **serious legal consequences**.

This repository is intended **strictly for educational and ethical cybersecurity research purposes**.
